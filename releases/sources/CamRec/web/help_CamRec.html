This section allows you to configure the CamRec function.

Configure CamRec

"Status" section show current state of ffmpeg-daemon.

Use button Stop for kill ffmpeg-daemon and stop recording.

Use button Start to start recording.

"Config" section allows you to configure ffmpeg-daemon.

Fill "Output directory" cell to set or change directory where videos will be saved (for example: /mnt/HD/HD_a2), and press "Save" button to save changes.

Fill "Host" cell to set or change the host from wich streaming (for example: 192.168.1.52), and press "Save" button to save changes.

Fill "Port" cell to set or change the port of host from wich streaming (for example: 7776), and press "Save" button to save changes.

Change "FPS" cell to set or change the Frames per Second of video-stream (for example: 10), and press "Save" button to save changes.

Change "Record time" cell to set or change the time(in minutes) of video-recording (for example: 30), and press "Save" button to save changes.

Change "DeInterlacing filter" cell to set or change the filter of video-recording (for example: Yadif), and press "Save" button to save changes.

Change "Store videos" cell to set or change the time(in days) for save videos (for example: 3), and press "Save" button to save changes.
